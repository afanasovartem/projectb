Исходный датасет содержит информацию о книгах.
В нём есть такие столбцы как год написания, количестов страниц, стоимость, автор, описание, подзоголовок, название редакции, язык, категории, средний рейтинг, число отзывов, ссылка и другие менее важные данные.
Изначально датасет имеет 15147 строк и 21 столбец
В ходе обработки при помощи pandas мы заменили пропуски на медианные занчения и unknown, удалили дубликаты и помяенили типы данных. Так же мы вывели статистику данных при помощи графиков и провели группировки по разным принцыпам.
Далее я сохранил обработанный файл в csv формате и использовал его для чартов в яндекс дата ленсе.
